# 全页面布局原型与结构说明

## 通用页面结构

```text
┌──────────────────────────────────────────────────────────────┐
│ 顶部：Sheet 名称 / 数据周期 / 页面类型 / 行数                  │
├──────────────────────────────────────────────────────────────┤
│ KPI 卡片区：Top5 核心指标                                     │
├──────────────────────────────────────────────────────────────┤
│ 维度筛选区：月份 / 加盟商 / 区域，侧边栏全局生效               │
├──────────────────────────────┬───────────────────────────────┤
│ 排名分析 TOP10               │ 倒数/异常对比                  │
├──────────────────────────────┼───────────────────────────────┤
│ 指标区间分布                 │ 收入/成本/补贴/贡献结构         │
├──────────────────────────────────────────────────────────────┤
│ 公斤段 / 状态 / 区域等动态模块                                │
├──────────────────────────────────────────────────────────────┤
│ 异常数据预警清单                                               │
├──────────────────────────────────────────────────────────────┤
│ 明细数据表：搜索 / 分页 / 排序 / 导出 CSV                      │
└──────────────────────────────────────────────────────────────┘
```

## 1. 总表-加盟商

核心模块：

- 加盟商经营贡献 TOP10/倒数10
- 核心指标分布，优先使用贡献、单量、重量等字段
- 收入、补贴、扣款、派费、贡献结构拆解
- 按加盟商维度支持筛选和明细下钻

## 2. 总表-网点

核心模块：

- 网点经营贡献 TOP10/倒数10
- 所属加盟商经营分布
- 在网、启用、测试运行等状态分析
- 区域维度分布分析

## 3. 出港考核、派费补贴

核心模块：

- 发生金额总览
- 加盟商/网点金额排名
- 政策名称维度筛选和明细查询
- 零值、负值、P99 高值预警

## 4. 派费分析报表

核心模块：

- 派费收入/支出结构
- 单票派费、发件/派件量相关指标分布
- 加盟商派费排名
- 超阈值、零派费、负派费异常清单

## 5. 一口价明细_达量一口价导出

核心模块：

- 一口价明细金额、票数、重量总览
- 所属加盟商排名
- 返利公斤段分析
- 按加盟商、公斤段、业务月份筛选明细

## 6. 总表-一口价

核心模块：

- 单量、计费重量、四费、一口价结算价、贡献总额 KPI
- 加盟商一口价贡献排名
- 单票贡献、单公斤贡献分布
- 经营结构拆解

## 7. 辽宁区域贡献

核心模块：

- 区域整体经营大盘
- 目的省份/区域贡献排名
- 公斤段结构分析
- 出港/进港、收入/成本/贡献结构拆解

## 8. 加盟商贡献

核心模块：

- 加盟商贡献排名与占比
- 目的省份、公斤段拆解
- 贡献、单量、重量、派费结构
- 明细下钻

## 9. 包仓费明细

核心模块：

- 包仓费汇总
- 加盟商/网点包仓费排名
- 零值、负值、高值预警
- 明细导出

## 10. 中心代建包费明细

核心模块：

- 总包数、总票数、计费重量、建包费 KPI
- 操作网点/寄件网点排名
- 费用为 0 票数预警
- 明细导出

## 11. 运营管理类汇总表

核心模块：

- 发生金额汇总
- 费用主类型、类型维度分析
- 网点/结算对象排名
- 负值、高值预警

## 12. 匹配表

基础档案框架：

- 数据总量、字段数量
- 公斤段映射完整性
- 空值、重复值质量校验
- 全量明细表

## 13. 网点资料

基础档案框架：

- 网点总量、有效数据量、状态字段质量
- 管理大区、行政大区、网点类型、启用状态分析
- 空值、重复值、关键字段缺失校验
- 全量档案表

## 14. Sheet2

基础档案框架：

- 网点-加盟商匹配关系总览
- 重复匹配、空匹配质量校验
- 全量关系表

## 15. Sheet3

基础档案框架：

- 加盟商清单总数
- 重复加盟商、空加盟商质量校验
- 全量加盟商清单
