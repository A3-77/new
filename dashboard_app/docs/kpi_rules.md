# 核心指标计算规则

## 1. 通用指标口径

数据模型层根据字段名称自动选择计算口径。

| 指标类型 | 识别关键词 | 计算规则 |
|---|---|---|
| 规模量 | 单量、票数、出港、进港、寄件签收量、派件量、总包数 | 求和 |
| 重量 | 重量、计费重量、总重量 | 求和 |
| 收入 | 收入、结算价、面单费、四费、应收 | 求和 |
| 成本/派费 | 派件费、派费、中转费、操作费、建包费、包仓费、成本 | 求和 |
| 补贴/返利 | 补贴、返利、加收 | 求和 |
| 扣款/考核 | 扣款、考核、罚款 | 求和 |
| 经营贡献 | 贡献、贡献总额、净贡献、进出港贡献 | 求和 |
| 均值类 | 均重、日均、单票、单公斤 | 平均 |
| 比率类 | 占比、达成率、增长率 | 平均 |

## 2. KPI Top5 选择规则

每个 Sheet 自动识别数值字段，并按物流加盟商经营优先级排序。

优先级：

1. 贡献总额、经营贡献、净贡献、进出港贡献
2. 单量、票数、寄件签收量、发件票数、派件量
3. 出港、进港
4. 重量、计费重量、均重
5. 面单费、派件费、派费
6. 补贴、扣款、发生金额
7. 收入、成本、四费

页面 KPI 卡片展示排序前 5 个字段。

## 3. 目标达成率

如果当前 Sheet 中存在字段名包含 `目标` 或 `指标` 的数值字段，则：

```text
目标达成率 = 当前指标汇总值 / 目标或指标字段汇总值
```

如果目标字段不存在，页面不强行展示达成率，避免生成无来源指标。

## 4. 排名分析

排名分析按当前页面可识别的最细经营主体进行聚合：

```text
主体指标值 = sum(主体下所有明细行的核心指标)
```

主体选择优先级：

1. 加盟商
2. 网点
3. 区域
4. 其他可识别维度

TOP10 按指标值降序，倒数10 按指标值升序。

## 5. 经营结构拆解

经营结构按字段关键词归集：

```text
收入 = sum(收入、结算价、面单费、应收、四费)
派费/成本 = sum(派件费、派费、中转费、操作费、建包费、包仓费、成本)
补贴 = sum(补贴、返利、加收)
扣款/考核 = sum(扣款、考核、罚款)
贡献 = sum(贡献相关字段)
```

字段匹配不到时，该结构项显示为 0，不从其他 Sheet 补数。

## 6. 区间分布

核心指标分布使用等宽分箱：

```text
区间数量 = min(8, 当前指标唯一值数量)
```

用于识别加盟商/网点的规模分层、极端值和经营集中度。

## 7. 异常预警

异常清单按当前 Sheet 内的数值字段识别：

- 金额、费用、贡献、补贴、扣款、考核类字段小于 0
- 金额、费用、贡献、补贴、扣款、考核类字段等于 0
- 任一数值字段超过当前字段 P99 分位

预警结果只作为运营排查清单，最终口径以业务规则确认后的阈值为准。
